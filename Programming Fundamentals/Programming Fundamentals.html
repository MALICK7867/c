Introduction To Programming

What is a Computer

Common Operating Machine Purposly Used for Technology, Education and Research is known as a COMPUTER. A computer is a electronic device which takes input from a user and then process this data into information and save or store the information in its storage devices and provide us with result when desired.

  • Takes Input
  • Process
  • Store
  • Produce Output

What is Program

A program is a set of instructions given to a computer to perform a specific task. For example, sorting numbers, calculating grades are basic examples of a program. A process of writing code in order to give instructions to a computer is called Programming.

Programming Language

A programming language is a notational system for describing instructions for computer in a machine-readable and human-readable form. There are many programming languages such as C, C++, Java, Python, Assembly etc. However, each programming language has its own programming rules, sytax and structures.

Levels of Programming Languages

Low Level Language

The programming languages which are easily understandable to machines but difficult to understand for human. There are two types of low level programming. ie.

  • Machine Language
  • Assembly Language

Machine Language

Machine Language consist of binary numbers (1,0). It is difficult for a human being to write codes in machine language. Program writing, reading, debugging and upgrading in machine language is very difficult for human. Therefore the programming in Machine Language is too slow.

Assembly Language

Assembly language uses symbolic codes which easy to remember such as ADD, SUM, MOV etc. Theses symbolic codes are developed against some binary codes. So writing, reading, debugging and upgrading the programs in Assembly language become easy as compare to the Binary Language.

High Level Language

To speed up the programming process High level Languages are developed. These languages are human friendly because these are more close to the human language. High-level languages, such as C, C++, Java, C#, Swift and Visual Basic, allow to write instructions that more like English and mathematical notations. However, High Level Languages are NOT directly understandable by the machines. They need to be translated into machine level. These languages are machine independent; i-e portable.

Language Processor

A language processor is a type of software program that translates the High Level Language program (source code) into Machine Language (object code). Language processor is also called language translator.

  • Source Code: The program written by the user.
  • Object Code: The program translated into machine language.

Compiler

Compiler converts program written in a high-level programming language into the machine code, at once, before program runs.

Interpreter

Interpreter converts each high-level program statement, one by one, into the machine code, during program run. Scripting languages JavaScript and PHP are processed by Interpreters.

Assembler

A type of converter that converts Assembly Language program into machine code.

Types of Errors

What is an Error

Programming errors are referred to as mistakes or bugs in the program. There are three types of errors:

  1. Syntax Error
  2. Runtime Error
  3. Logical Error

Syntax Error

A type of error that occurs at the time of compilation. It usually occurs due to an error in the rules & regulation or structure of the program. It do not allow the program to get compiled and as a result not result in execution of program. It is very easy to find because compiler programs usually highlight the error.

Runtime Error

A type of error that occur at the time of execution. Occurs due to an illegal operation in the program such as dividing by zero, referencing an invalid file, calling invalid function. It cause the program to stop or crash. It is diffult to identify as compared to Syntax error.

Logical Error

A type of error that occur at the time of execution. It usually occur due to incorrect logic or fault in the algorithm in the program such as incorrect sequence of instuctions. It do not stop the compilation or execution of the program but gives wrong output of the program. It is very hard and complex to identify.

Life-Cycle of Program

Introduction

Program lifecycle consist of the stages in which a program undergoes from initial creation to deployment and execution. These phases consist of:

  • Program Editting
  • Pre-Processing
  • Compilation
  • Linking
  • Loading
  • Execution

Full Lifecycle

  • Program Editing: Program is created in the editor and stored on disk as .cpp file
  • Preprocessing: Preprocessor program processes the code
  • Compilation: Compiler creates object code and stores it on disk as .obj file
  • Linking: Linker links the object code with the libraries and stores as .exe file
  • Loading: Loader puts program in memory
  • Execution: CPU takes each instruction and executes it, possibly storing new data values as the program executes

Datatypes

Variables

During programming we need to store data like marks, names, etc. To store these values we use variables. These variables are located in the memory of system. This memory is divided into blocks and each block has its own unique numberical address. To store and access these blocks of the memory, we give them a name and these are called variables. These are called variables because the value is different or may change and store different values at different time. These variables are declared using these rules:

  • Type of variable
  • A name of variable
  • Size of variable
  • Value stored in variable

Variable Type

A variable must have a datatype associated with it. It is used to identify which type of data is stored in a variable. For example:

  • Integer (used for storing whole numbers)
  • Float (used for storing numbers, decimal points)
  • Character (used to store alphabets)
  • Boolean (true/false)
  • String (used to store text)

Variable Size

Different variables have different size. It is a datatype which tells us about the size of the variable. It depends on the IDE to assign size to a specific datatype. Here, we will discuss the size of different variables used in Dev C++. ie.

  • Integer (int) = 4 Bytes
  • Float (float) = 4 Bytes
  • Double (double) = 8 Bytes
  • Character (char) = 1 Byte
  • String (string) = 8 Bytes
  • Boolean (bool) = 1 Byte
