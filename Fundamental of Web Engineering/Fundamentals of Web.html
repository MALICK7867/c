Media Tags

Video Tag

A tag which is specifically used to add video in an HTML page. There are some attributes that can be added to video player in browser player. These are:

  • Controls: It adds controls to the player.
  • Autoplay: Makes video play automatically when page is loaded.
  • Loop: Make video play in loop.
  • Muted: Play video without sound by default.
  • Poster: Apply a thumbnail to a video.

<video src="source.mp4" controls autoplay loop muted poster="source.png"></video>

Audio Tag

A tag which is specifically used to add audio in an HTML page. There are some attributes that can be added to audio player in browser player. These are:

  • Controls: It adds controls to the player.
  • Autoplay: Makes audio play automatically when page is loaded.
  • Loop: Make audio play in loop.
  • Muted: Make audio play without any sound.

<audio src="source.mp3" controls autoplay loop muted></audio>

Youtube Embeding

Youtube provides a feature in which a youtube video can be played in your HTML page. This feature is known as youtube embeding. YouTube provides a link of that video which will be played on your HTML page. As mentioned earlier, you can add attibutes to youtube embedded video just like a normal video but there are some other ways to add attributes in this type of videos. ie.

  • ?autoplay = 1 or 0
  • &Loop = 1 or 0
  • &Controls = 1 or 0
  • &Muted = 1 or 0

<iframe width="560" height="315" src="https://www.youtube.com/embed/dYH6FpmMsus?si=xWONsv29Qi5v1TsF" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>
